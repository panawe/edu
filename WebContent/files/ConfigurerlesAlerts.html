Configurer les Alerts

Il existe trois types d'alertes.


  • Retard de retour d'un emprunt: Lorsqu'un emprunt depasse la date prevu pour le retour,
    cet alerte est genere pour rappeler a l'emprunteur d'honnorer son engagement.
  • Quantite minimum de stock atteint: Lorsque le stock minimum d'un produit est atteint,
    un e-mail est envoye aux responsables de l'etablissement pour qu'ils reconstitue le stock.
  • Retard de payment: Lorsque la date limite de payement d'une tranche est atteinte,

     cet alerte envoie un e-mail au etudiants pour leur rappeler le payement le delai et le montant de payement.


Le contenu du message peut etre change a volonte et une liste de mots clefs est disponible pour permettre

de specifier les valeurs dynamiques telles par example le montant restant a payer. Voici liste des mots clefs.

Plusieurs parametres permettent de controler le fonctionnement des alertes.


  • Nombre de fois: Defini le nombre de fois que le message d'alerte est envoye aux interesses
  • Interval de repetition: Defini l'interval entre les envois d'e-mails de rappel
  • Envoyer a: Liste des e-mails auxquels il faut envoyer l'alerte. Dans le cas des retards de payement,
    un e-mail est automatiquement envoye aux etudiants concernes en plus de la liste des e-mails dans ce champs.
    Les e-mails sont separes par un ";".
  • Afficher?: Ce champs permet de decider d'afficher ou non les alertes sur la page d'accueil de l'etudiant.
  • Nombre de jours a afficher: Definit le nombre de jours que le message d'alerte est affiche dans sur la page d'accueil de l'etudiant
  • Nombre de jour: Nombre de jours apres ou avant que le delai soit expire pour que l'alerte soit genere. Ce parametre fonctionne de paire
    avec le parametre "Moment". Ce parametre n'est applicable qu'aux retards de payment et de retour d'emprunt.
  • Moment: Determine si l'alerte est genere avant ou apres la date d'expiration. Par exemple si le champ "Nombre de jours" est fixe a 4 et le moment
    est defini comme "Apres" alors l'alerte est genere 4 jours apres que delai soit expire. L'autre valeur du champs "Moment" est "Avant".

Il permet de generer l'alerte a l'avance.
