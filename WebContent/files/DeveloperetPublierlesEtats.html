Developer et Publier les Etats

De nouveaux etats peuvent etre integres a l'application sans effort supplementaire de developement autre que le developement de l'etat lui-meme.

Pour ce faire, les conditions suivantes doivent etre remplies:


  • L'etat est develope avec la version 3.7.6 de l'outil iReport
  • Les parametres utilises dans l'etat respectent la nomenclature et sont strictement choisis dans la liste du tableau ci-dessous.
  • Un menu est ajoute a travers la page de gestion de la securite de l'application
    • Le nom du menu importe peu mais il est recommende qu'il reflecte le contenu de l'etat
    • Le URL du menu doit etre dans ce format: dispatch?link=report&reportName=resultatTrimestriel
      • Seule la partie du URL en rouge change
      • Cette partie correspond au nom de l'etat. Example: si l'etat s'appelle resultatTrimestriel.jasper alors le URL est dispatch?link=report&reportName=resultatTrimestriel
  • L'etat doit etre deploye dans le repertoire des etats de l'application.
  • Une fois que les contraites ci-haut son respectees, l'application construit automatiquement la page de parametres lorsqu'on y accede.
