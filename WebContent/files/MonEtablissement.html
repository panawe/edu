Mon Etablissement

Cette page permet de configurer les parametres generaux specifiques a un etablissement.


Parametres d'un etablissement


  • Nom: Il s'agit du nom de l'etablissement
  • Nom Abbrege: Le nom abbrege est utilise sur certains etats du fait que le nom de l'etablissement peut etre long
  • Entete du bulletin: Le contenu de ce champ est affiche sur l'entete du bulletin ainsi que sur celui des autres etats.
  • Afficher l'entete du bulletin: Si cette case est cochee, l'entete contenant le logo est affiche sur le bulletin.
    Cela permet d'imprimer le bulletin sur du papier ayant deja un entete.
  • Afficher le mot de passe par defaut: Si cette case est cochee, le mot de passe par defaut est affiche sur la fiche d'inscription de l'etudiant.
  • Generer le mot de passe par defaut: Si cette case est cochee, le mot de passe par defaut est automatiquement genere par le systeme lorsque l'etudiant est ajoute la base de donnees.
  • Logo: Le logo est affiche aussi bien sur l'entete du bulletin ainsi que sur le coin haut et gauche du site web de l'ecole.

Classes de l'etablissement

Plage d'appreciation des notes.


Cette page permet de definir la plage de notes correspondantes a chacune des appreciations.

Les appreciations sont automatiquement determinees par le systeme lorsque la note est saisie.
